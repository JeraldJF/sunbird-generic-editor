# Content Editor Project

This project is designed to build the content editor using GitHub Actions. It includes a workflow that automates the build process, ensuring that the latest changes are integrated and artifacts are generated.

## Directory Structure

- `.github/workflows/build.yml`: Defines the GitHub Actions workflow for building the content editor.
- `.github/scripts/build.sh`: Shell script that contains the commands to build the content editor.
- `.github/README.md`: Documentation for the project.

## Usage

To trigger the build process, push changes to the repository or create a pull request. The GitHub Actions workflow will automatically run, executing the defined steps to build the content editor and archive the artifacts.
